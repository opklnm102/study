## zsh r command
```sh
$ r
```
* `!!`, `fc -e -`와 동일한 명령어로 마지막 명령어를 실행


<br>

## tail -f *.log
```sh
$ tail -f *.log
```
* 해당 경로에 있는 모든 로그 파일을 파일명으로 구분해서 추적하므로 여러 창에서 각 파일을 추적할 필요가 없다
